The AI agent that made me delete OpenClaw: NanoClaw x Venice
I FINALLY found the AI agent setup I've been looking for and I can't stop tinkering with it
Recently I had a horrible of week of banging my head against the wall with Frankenstein monster OpenClaw. I couldn't get OpenClaw to work on the most basic stuff, and I thought it was a skill issue
So I decided to branch out and try something else: NanoClaw
Played around with it for a couple days and realised THIS is the AI agent I've been waiting for all my life. Setup was super easy and fast, bug fixes are satisfying (you get help from Claude Code), and the agent is SNAPPY af in my Whatsapp.
It just works. But there was one thing gnawing at me: I was running it through Anthropic API, and you know they be spyin' on my shit
So I forked the repo and rebuilt it to run through the
Venice API
to anonymously use Claude Sonnet 4.6 and Claude Opus 4.6 (or fully private if you use the open-source models)
Repo here:
github.com/lorenzovenice/nanoclaw-venice
What you get out of the box:
  • NanoClaw agent on WhatsApp and/or Telegram
  • Claude Code CLI and agent runs through the Venice API
  • Everything gets its own isolated Docker container with separate memory
  • Easy model switching in chat ("switch to opus", "use glm-5")
  • Cron jobs, scheduled tasks and reminders etc.
  • Web search and browsing built in
That's it, you say? Yeah, that's it. Light, breezy, secure. But you can do so much more with it and customize it to your liking.
Why this is better than OpenClaw (esp for noobs like me):
  • NanoClaw is ~2k lines of core code vs OpenClaw's ~500k. Not a framework, just a clean lightweight bot.
  • New capabilities are added as skills through the Claude Code CLI, so it's fully modular. You only add what you need, nothing bloated out of the box.
  • Way easier to debug, customize, and actually understand what's going on and where to fix. Something breaks, just ask Claude Code.
  • Adding skills and making fixes feels like a videogame instead of purgatory like OpenClaw, the dopamine release is unreal
I love it so much, a no-coder like is making PRs to the official repo. I even got a like on my
NanoClaw PR
from the man himself.
Heads up:
  • There's an detailed
    README.md
    that takes you through all the steps from the POV of a no-coder / beginner vibe coder
  • If the setup wizard doesn't progress automatically, just say "next step" int the Claude Code CLI
  • If something breaks during setup or after, just use the Claude Code CLI to debug. It's really great at that and has access to your entire build
  • Check out the extensive
    troubleshoot
    section, I went through all that pain for y'all
If anything goes wrong, let me know or make a PR :)
